# Galary App Using React

# Installation

- Clone the repo `git clone https://github.com/alaspuresujay/react-galary.git`
- In the repo directory, run to install dependencies
  - `npm install`
- `npm start`

## Technology

- React
- Firebase (Clone Database)
- framer-motion
